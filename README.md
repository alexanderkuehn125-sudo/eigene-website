# Eigene Website

Willkommen zu deinem Website-Projekt!

## Entwicklungsserver starten

Um die Website lokal anzuschauen, musst du den Entwicklungsserver starten. Öffne dazu ein Terminal in diesem Ordner und führe folgenden Befehl aus:

```bash
npm run dev
```

## Lokaler Link

Sobald der Server läuft, kannst du deine Website jederzeit unter diesem Link im Browser aufrufen:

**[http://localhost:8080](http://localhost:8080)**
